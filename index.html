PROJECTS

POKESEARCH

-------------------

PokeSearch is a sleek and interactive web application that allows users to search for Pokémon by name and filter them by their type and instantly view information such as their images, types,and Id no. Built using HTML, CSS, and JavaScript,the app utilizes the PokeAPI to fetch real-time data. With a responsive design and a clean user interface,PokeSearch delivers a smooth and engaging user experience for Pokémon fans. This project reflects my ability to work with external APIs, manage asynchronous JavaScript (fetch/async/await), and build dynamic, user-focused interfaces.

SPOTIFY REPLICA

-----------------

I designed a static replica of Spotify’s user interface using only HTML and CSS, inspired by a YouTube tutorial since it was my first project. The layout features a sidebar menu, a trending music section, and a bottom media player, closely mimicking Spotify’s structure and visual design. This project helped me practice responsive layouts, positioning, and modern UI styling, all without using JavaScript.

SIMON SAYS GAME

-----------------

Simon Says Game is a classic memory-based game that I recreated using pure HTML,CSS, and JavaScript. In this interactive project, players must memorize and repeat a growing sequence of colored button flashes. Each level adds a new color to the pattern, testing the user’s memory and attention. The game uses dynamic DOM manipulation, animations, and basic game logic to deliver an engaging experience. It also includes a scoring system and restart functionality upon a wrong move
